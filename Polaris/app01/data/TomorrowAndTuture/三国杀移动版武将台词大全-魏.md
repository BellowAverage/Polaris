哈哈，今天来干一件傻事，来写写三国杀移动版武将台词大全（写之前也搜了一些，然而所谓大全其实并不全，终究还是错付了，所以不如自力更生），本来准备用 json 格式，但是 yaml 其实看起来更简洁一些（毕竟少了 json 当中的大括号{}，中括号[] 和引号 ""，这也是它比较受欢迎的原因之一），其实主要是为了写起来也方便，不信的话可以尝试转换后对比看一下就知道了：

```
require 'yaml'
require 'json'

File.open("sgs.json", "w") {|file| file.puts JSON.pretty_generate(YAML.load(File.open("sgs.yaml")))}
```

使用 Python，Ruby 等之类的读取成 Hash 的形式也很方便，对于学习 yaml 配置文件格式，也是很有帮助的哟。因为武将比较多，每个武将的皮肤也比较多，所以台词也就比较多了（收集的工作量还是挺大的），这个就只能后边有时间慢慢补充了。如果你喜欢玩三国闲话里边的“武将猜猜猜”活动，这绝对是不可多得的秘籍哟！**若有更新遗漏之处，还望诸位指点一二！**

因博客字数限制，遂将武将台词按照势力拆分到不同博客。拆分code如下，大家有兴趣也可以自己聚合：

```
require 'yaml'

comment_lines = File.read("sgs.yaml").lines
comment_lines.keep_if { |line| line.include?("新增") }

result = YAML.load(File.open("sgs.yaml")) # sgs.yaml 全部势力武将台词
["魏", "蜀", "吴", "群", "神"].each do |index|
  tmp = ""
  peoples = result[index].keys.sort
  lines, res = peoples.size / 16, peoples.size % 16
  if res &gt; 0
    lines += 1
  end
  comment_lines.each do |line|
    if peoples.any?{|people| line.include?(people)}
      tmp &lt;&lt; line
    end
  end

  tmp &lt;&lt; "# #{index}（#{peoples.size}/#{peoples.size}）：\n"
  lines.times do
    tmp &lt;&lt; "#     #{peoples.pop(16).join(" ")}\n"
  end

  data = result[index]
  File.open("sgs_#{index}.yml", "w") do |file|
    file.write("# sgs_#{index}.yml\n\n")
    file.write(tmp)
    file.write(YAML.dump(data))
  end
end
```
