## Attempt to call an undefined function glutInit



解决方法：

从这里下载PyOpenGL 的whl安装文件，





安装命令举栗

pip install PyOpenGL-3.1.7-cp39-cp39-win_amd64.whl 



pip install PyOpenGL_accelerate-3.1.7-cp39-cp39-win_amd64.whl
