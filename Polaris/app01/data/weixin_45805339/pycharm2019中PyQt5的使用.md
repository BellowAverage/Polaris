## pycharm2019中PyQt5的使用

### 1.PyQt5安装

```
pip install PyQt5 -i https://pypi.tuna.tsinghua.edu.cn/simple

```

```
pip install PyQt5-tools -i https://pypi.tuna.tsinghua.edu.cn/simple

```
