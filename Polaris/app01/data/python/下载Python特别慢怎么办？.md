一般我们下载python都是去官网（）这个网址去下载的，但是下载速度特别特别慢，哪怕家里是千兆网也要20分钟（反正我家是这样），那有什么好方法来解决这个问题吗？

有，既然在官网下载慢，那我们就去镜像站下载不就好了嘛，国内有很多镜像站，像什么阿里云，腾讯云，清华镜像站等等，但是这些镜像站一般人玩不明白，所以这里推荐淘宝镜像站然后进去以后找到python
