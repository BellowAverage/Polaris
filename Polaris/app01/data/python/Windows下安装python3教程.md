## 一、安装步骤图解

**准备工作：**

进官网下载安装包，注意：Python 3.9+不能在Windows 7或更早版本上使用<img alt="" height="734" src="https://img-blog.csdnimg.cn/0387df2395464ed5af6d362fb28357f0.png" width="1200">

**安装：** 

1.下载完之后双击该文件，会弹出以下框框，勾选最后一项 **Add python.exe to PATH** ，把python添加到环境变量PATH中，再选择自定义安装  **Customize installation** 

<img alt="" height="412" src="https://img-blog.csdnimg.cn/bd7e5581cb98471bbffc1d7ceaad2dcd.png" width="670">

2.点击**Next**

<img alt="" height="412" src="https://img-blog.csdnimg.cn/e06c061f1a864f3eaacf799100e4cbf5.png" width="670">

3.勾选**Install Python 3.11 for all users**和**Download debugging symbols，**自定义安装路径，这里我选择的是D盘

<img alt="" height="412" src="https://img-blog.csdnimg.cn/12099b0df9a24ec49047129293a51409.png" width="670"> 4.点击**Install**开始安装

<img alt="" height="412" src="https://img-blog.csdnimg.cn/3e5b02989c27445ea76c35d9096e49f5.png" width="670">

5.安装完成，点击 **Close **

<img alt="" height="412" src="https://img-blog.csdnimg.cn/42ab251e587e4e9eafe2ceace51ffc96.png" width="670">
