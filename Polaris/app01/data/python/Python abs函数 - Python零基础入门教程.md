### 一.Python abs 函数介绍

 函数是  内置函数，主要作用就是计算数字的绝对值！语法如下：

```
'''
参数x：只能为有符号的数字，例如：0，5，6.3，-78……
返回值：直接取当前数字的绝对值，必然大于等于0，不会小于0;
'''
abs(x)

```
