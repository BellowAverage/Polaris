#### Pycharm支持的Python版本



一直用的老版本Pycharm（2018）和Python（3.6），最近需要用到python 3.10，电脑安装后发现配置Pycharm编译器时报[unsupported]，查了下是因为Pycharm版本低导致，扒了下Pycharm，整理出各版本Pycharm和Python的对应关系，供参考~
