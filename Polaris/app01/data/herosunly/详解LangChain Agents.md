## 1. 前言

  随着大模型技术的逐步发展，以LLM(大型语言模型)为核心构建agent已经成为研究的热点。OpenAI应用研究主管LilianWeng曾提出了Agent = LLM + memory + planning skills + tool use，也就是智能体=大模型+记忆+规划+工具使用。
