python编程语言中存在着很多的便捷方法，如本次我们要学习的常规多变量一次赋值不同类型的方法，就是其中之一，通过本书写方法可以精简代码，利用一行直接赋值几十个上百个变量。

## 代码实例：

```
# 一次性赋值多个不同类型变量的代码实例
one,two,three,four,five = 'yi','二',['san','1',3],4,'五'
# 输出结果
print(one,two,three,four,five)

```
