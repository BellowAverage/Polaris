#### 搜索语法汇总【建议收藏】

>  
 **1、 filetype  **搜索文件的后缀或者扩展名 


```
filetype:

介绍：通过搜索引擎，获取指定类型的文件。

搜索示例:filetype:xls
将返回所有以xls结尾的excel文件的URL地址
```

 **2、info 网站基本信息** 


```
info

介绍：搜索指定网站的一些基本信息。

搜索示例：info：www.csdn.net

         获取csdn的相关信息

```
