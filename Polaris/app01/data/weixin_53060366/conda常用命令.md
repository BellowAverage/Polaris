### conda常用命令

```
查看帮助
conda -h

查看conda版本
conda --version

安装django包
conda install django

查看已经安装的包
conda list

更新包
conda update django

删除包
conda remove django

创建环境
conda create -n name python=version

激活环境
activate name

退出当前环境
conda deactivate

删除环境
conda remove -n name --all

查看所有虚拟环境
conda env list

```
